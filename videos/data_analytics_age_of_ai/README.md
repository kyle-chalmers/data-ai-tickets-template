# What Is Data Analytics in the Age of AI?

Companion materials for the KC Labs AI video exploring how AI is transforming data analytics -- from the roles that power it to the tools reshaping every step.

[Watch on YouTube](https://www.youtube.com/@kylechalmersdataai) | [KC Labs AI](https://kclabs.ai)

## What You'll Learn

1. What data analytics is and how AI is transforming it, with real examples from Spotify, Starbucks, and Netflix
2. The 4 key data roles (Data Engineer, Data Analyst, Analytics Engineer, BI Developer) and how AI is augmenting each one
3. Why human judgment is the most important skill in the age of AI, and why these careers are growing, not shrinking

## Try It Yourself

The video includes a live demo where we paste a simple dataset into Claude (free) and ask it to recommend which city a coffee chain should expand into.

**Want to try it?**

1. Copy the dataset from [sample_data/coffee_chain_locations.csv](./sample_data/coffee_chain_locations.csv)
2. Go to [claude.ai](https://claude.ai) (free account)
3. Paste the CSV and ask: *"Based on this data, which city should we open our next coffee shop in and why? Consider revenue potential, foot traffic, rent costs, competition, and market size."*

Other free AI tools that can do this:

- [ChatGPT](https://chat.openai.com)
- [Gemini](https://gemini.google.com)

## The 4 Data Roles

These four roles form the backbone of data analytics teams. AI is changing how each one works, but it is not replacing any of them.

| Role               | Tagline                 | Tools                                    |
| ------------------ | ----------------------- | ---------------------------------------- |
| Data Engineer      | "Gets the data flowing" | Python, SQL, dbt, Spark, Cloud Platforms |
| Data Analyst       | "Finds the answer"      | SQL, Excel, Tableau, Power BI            |
| Analytics Engineer | "Makes it repeatable"   | dbt, SQL, Snowflake, BigQuery            |
| BI Developer       | "Makes it visible"      | Power BI, Tableau, SQL                   |
